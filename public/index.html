Zapatería Backend

API RESTful

ZAPATOS

GET /api/zapatos Obtener todos los zapatos.
POST /api/zapatos Añadir un zapato.
GET /api/zapatos/{id} Obtener un zapato.
UPDATE /api/zapatos/{id} Modificar un zapato.
DELETE /api/zapatos/{id} Eliminar un zapato.

EMPLEADOS

GET /api/empleados Obtener todos los empleados.
POST /api/empleados Añadir un empleado.
GET /api/empleados/{id} Obtener un empleado.
UPDATE /api/empleados/{id} Modificar un empleado.
DELETE /api/empleados/{id} Eliminar un empleado.
